How it works

  • Collapsed: Thin bar showing title + page bookmark — one click to expand
  • Expanded: PDF rendered via browser-native PDF.js, white page on dark background
  • Page nav: Prev/Next arrows + keyboard shortcuts (arrow keys when focused)
  • Zoom: +/- buttons, fits to width by default
  • Bookmark: Star icon saves current page — auto-restores on next session
  • Empty state: Dashed border with "Upload PDF" and "Library" buttons for quick attach
